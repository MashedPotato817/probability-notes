🎲 一、概率基础

1.1 核心公式

公式名称表达式适用条件
加法公式$P(A \cup B) = P(A) + P(B) - P(AB)$任意两事件
条件概率$P(A|B) = \frac{P(AB)}{P(B)}$$P(B) > 0$
乘法公式$P(AB) = P(A) \cdot P(B|A)$链式法则
全概率公式$P(A) = \sum_{i} P(B_i) P(A|B_i)$$B_i$ 为完备划分
Bayes公式$P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_j P(B_j)P(A|B_j)}$逆概率

1.2 事件关系

对立事件

$A \cup B = \Omega$

$AB = \emptyset$

$P(A) = 1 - P(B)$

互不相容

$AB = \emptyset \Rightarrow P(AB) = 0$

独立

$P(AB) = P(A) \cdot P(B)$

互不相容 ≠ 独立：若 $P(A)>0, P(B)>0$，互斥时 $P(AB)=0 < P(A)P(B)$，反而不独立！

1.3 古典概型

$P(A) = \frac{\text{有利事件数}}{\text{总事件数}}$
  • 排列: $A(n,m) = \frac{n!}{(n-m)!}$
  • 组合: $C(n,m) = \frac{n!}{m!(n-m)!}$

📊 二、随机变量分布

2.1 分布函数性质（必背）

  1. $0 \le F(x) \le 1$
  2. 单调不减（不是严格增！）
  3. 右连续: $F(x^+) = F(x)$
  4. $P\{X = x\} = F(x) - F(x^-)$

2.2 常见分布期望方差（必背核心表）

分布记号期望 $E(X)$方差 $D(X)$
0-1分布$B(1,p)$$p$$p(1-p)$
二项分布$B(n,p)$$np$$np(1-p)$
泊松分布$P(\lambda)$$\lambda$$\lambda$
均匀分布$U[a,b]$$\frac{a+b}{2}$$\frac{(b-a)^2}{12}$
指数分布$Exp(\lambda)$$\frac{1}{\lambda}$$\frac{1}{\lambda^2}$
正态分布$N(\mu, \sigma^2)$$\mu$$\sigma^2$
伽马分布$\Gamma(\alpha, \beta)$$\alpha\beta$$\alpha\beta^2$
指数分布参数注意：率参数 $\lambda$ 时 $E(X) = \frac{1}{\lambda}$；尺度参数 $\theta$ 时 $E(X) = \theta$！

2.3 随机变量函数分布

求分布函数: $F_Y(y) = P\{Y \le y\} = P\{g(X) \le y\}$
求导得密度: $f_Y(y) = \frac{dF_Y(y)}{dy}$

📈 三、二维随机向量

3.1 联合分布与边缘分布

边缘分布（离散型）

$P(X = x) = \sum_y P(X=x, Y=y)$

边缘密度（连续型）

$f_X(x) = \int_{-\infty}^{+\infty} f(x,y) \, dy$

独立时，联合密度 = 边缘密度乘积: $f(x,y) = f_X(x) \cdot f_Y(y)$

3.2 独立性判定

离散型

$P(X=x, Y=y) = P(X=x) \cdot P(Y=y)$

对所有 $x, y$ 成立

连续型

$f(x,y) = f_X(x) \cdot f_Y(y)$

对所有 $x, y$ 成立

只要有一点不满足就不独立！

3.3 二维正态分布

$(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$
  • $\rho = 0 \Leftrightarrow X, Y$ 独立（仅二维正态成立！）
  • $X, Y$ 的线性组合仍服从正态分布

📐 四、数字特征

4.1 期望性质

性质公式条件
常数期望$E(c) = c$无
线性性$E(aX + b) = aE(X) + b$无
可加性$E(X + Y) = E(X) + E(Y)$无条件成立
乘积期望$E(XY) = E(X)E(Y)$仅当独立或不相关
方差定义$E(X^2) = [E(X)]^2 + D(X)$方差展开

4.2 方差性质

性质公式
常数方差$D(c) = 0$
线性变换$D(aX + b) = a^2 D(X)$
和的方差$D(X + Y) = D(X) + D(Y) + 2Cov(X,Y)$
差的方差$D(X - Y) = D(X) + D(Y) - 2Cov(X,Y)$
关键结论: $D(X+Y) = D(X-Y) \Leftrightarrow Cov(X,Y) = 0 \Leftrightarrow$ 不相关

4.3 协方差

定义

$Cov(X,Y) = E(XY) - E(X)E(Y)$

性质
  • $Cov(X, X) = D(X)$
  • $Cov(X, Y) = Cov(Y, X)$
  • $Cov(aX, bY) = ab \cdot Cov(X, Y)$
  • $Cov(X + Y, Z) = Cov(X, Z) + Cov(Y, Z)$
  • $X, Y$ 独立 $\Rightarrow Cov(X,Y) = 0$
协方差和方差中加常数不影响，直接删除！

4.4 相关系数

定义

$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{D(X) \cdot D(Y)}}$

  • $-1 \le \rho \le 1$
  • $\rho = \pm 1 \Leftrightarrow Y = aX + b$ 完全线性相关
  • $\rho = 0 \Leftrightarrow$ 不相关
  • $\rho(aX+b, cY+d) = \text{sign}(ac) \cdot \rho(X,Y)$

4.5 不相关与独立的关系

关系结论
不相关定义$Cov(X,Y) = 0 \Leftrightarrow E(XY) = E(X)E(Y)$
独立 → 不相关✅ 成立
不相关 → 独立❌ 一般不成立
二维正态特例不相关 ⇔ 独立 ✅

∞ 五、大数定律与中心极限定理

5.1 大数定律

伯努利大数定律

频率收敛于概率: $\frac{S_n}{n} \to p$

辛钦大数定律

样本均值收敛于总体期望: $\bar{X} \to E(X)$

5.2 棣莫弗-拉普拉斯定理（二项分布中心极限定理）

设 $X \sim B(n,p)$，当 $n$ 充分大时：

$X \approx N(np, np(1-p))$

标准化: $\frac{X - np}{\sqrt{np(1-p)}} \approx N(0,1)$

5.3 独立同分布中心极限定理

$X_1, \ldots, X_n$ 独立同分布，$E(X) = \mu$, $D(X) = \sigma^2$:

$\sum_{i=1}^n X_i \approx N(n\mu, n\sigma^2)$

$\bar{X} \approx N(\mu, \frac{\sigma^2}{n})$

应用识别: 涉及"和"或"平均"时使用中心极限定理

🔬 六、抽样分布与统计量

6.1 三大分布（必背核心）

χ² 分布（卡方分布）

定义: $X_1, \ldots, X_n \sim N(0,1)$ 独立同分布

$\sum_{i=1}^n X_i^2 \sim \chi^2(n)$

性质:

  • $E(\chi^2(n)) = n$, $D(\chi^2(n)) = 2n$
  • 可加性: $\chi^2(n_1) + \chi^2(n_2) \sim \chi^2(n_1+n_2)$
  • 自由度 = 平方项数
t 分布

定义: $X \sim N(0,1)$, $Y \sim \chi^2(n)$, $X, Y$ 独立

$\frac{X}{\sqrt{Y/n}} \sim t(n)$

性质:

  • $E(t(n)) = 0$ (n>1)
  • 密度为偶函数，关于0对称
  • $n \to \infty$ 时趋于 $N(0,1)$
F 分布

定义: $X \sim \chi^2(n_1)$, $Y \sim \chi^2(n_2)$, $X, Y$ 独立

$\frac{X/n_1}{Y/n_2} \sim F(n_1, n_2)$

性质:

  • 先分子后分母写自由度
  • $F \sim F(n_1, n_2) \Rightarrow \frac{1}{F} \sim F(n_2, n_1)$

6.2 正态总体抽样分布定理（最核心！）

设 $X_1, \ldots, X_n$ 来自 $N(\mu, \sigma^2)$，样本均值 $\bar{X}$，样本方差 $S^2$:
结论分布
$\bar{X} \sim N(\mu, \frac{\sigma^2}{n})$正态
$\frac{\sqrt{n}(\bar{X} - \mu)}{\sigma} \sim N(0,1)$标准正态
$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$卡方(n-1)
$\bar{X}$ 与 $S^2$ 独立重要结论
$\frac{\sqrt{n}(\bar{X} - \mu)}{S} \sim t(n-1)$t分布
注意: 自由度是 $n-1$，不是 $n$！

6.3 样本均值方差的数字特征

  • $E(\bar{X}) = \mu$
  • $D(\bar{X}) = \frac{\sigma^2}{n}$
  • $E(S^2) = \sigma^2$（无偏估计）

🎯 七、参数估计

7.1 矩估计法

计算总体矩 $E(X)$（含参数 $\theta$）
令 $E(X) = \bar{X}$（样本矩代替总体矩）
解方程得 $\hat{\theta}$
K阶矩估计：令 $E(X^K) = A_K$（样本K阶矩）

7.2 最大似然估计（MLE）

离散型: 构造似然函数 $L(\theta) = \prod_{i} P(X_i)$
连续型: 构造似然函数 $L(\theta) = \prod_{i} f(X_i)$
取对数: $\ln L(\theta)$
求导: $\frac{d \ln L}{d \theta} = 0$
解出 $\hat{\theta}$
小写 $x$ 改大写 $X$ 得估计量（随机变量形式）

7.3 无偏性

定义: $E(\hat{\theta}) = \theta$
常用结论:
  • $\bar{X}$ 是 $\mu$ 的无偏估计
  • $S^2$ 是 $\sigma^2$ 的无偏估计
  • 样本方差 $\frac{1}{n}\sum(X_i - \bar{X})^2$ 是有偏估计

7.4 区间估计

σ 已知，μ 的置信区间

$\bar{X} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$

σ 未知，μ 的置信区间

$\bar{X} \pm t_{\alpha/2}(n-1) \cdot \frac{S}{\sqrt{n}}$

σ² 的置信区间

$\left[ \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)} \right]$

✅ 八、假设检验

8.1 检验步骤

建立假设: $H_0$（原假设） vs $H_1$（备择假设）
选择检验统计量
确定拒绝域
计算并判断

8.2 正态总体均值检验

条件检验统计量拒绝域（双侧）
$\sigma$ 已知$U = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$$|U| > z_{\alpha/2}$
$\sigma$ 未知$T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$$|T| > t_{\alpha/2}(n-1)$

📝 九、真题高频考点

📌 速记口诀
  • 矩估计: "总体矩=样本矩"
  • MLE: "似然→对数→求导→解方程"
  • 独立性: "联合=边缘乘积"
  • 相关性: "协方差=乘积期望-期望乘积"
  • 中心极限定理: "n大正态化"
  • χ²自由度: "平方项数"
  • 正态抽样: "X̄独立S²，(n-1)S²/σ²~χ²(n-1)"

易错点总结

  1. ❌ 分布函数是单调不减，不是严格增
  2. ❌ 互不相容≠独立（互斥时更不独立）
  3. ❌ 不相关推不出独立（除非二维正态）
  4. ❌ $E(XY) = E(X)E(Y)$ 仅当独立或不相关成立
  5. ❌ $D(X) = E(X^2) - [E(X)]^2$，不是 $[E(X)]^2$
  6. ❌ 样本方差 $S^2$ 自由度是 n-1，不是 n
  7. ❌ χ² 分布自由度 = 平方项数（不是样本数）
  8. ❌ 协方差和方差中加常数不影响，要删除
  9. ❌ 指数分布参数 $\lambda$ 和 $\theta$ 要注意
  10. ❌ MLE 求导后要验证极值点
