## 一、概率基础

### 1.1 概率公式

| 公式 | 表达式 | 说明 |
|------|--------|------|
| 加法公式 | P(A∪B)=P(A)+P(B)-P(AB) | 容斥原理 |
| 条件概率 | P(A|B)=P(AB)/P(B) | P(B)>0 |
| 乘法公式 | P(AB)=P(A)P(B|A) | 链式法则 |
| 全概率公式 | P(A)=ΣP(B_i)P(A|B_i) | 划分完备 |
| Bayes公式 | P(B_i|A)=P(B_i)P(A|B_i)/ΣP(B_j)P(A|B_j) | 逆概率 |

### 1.2 事件关系

- **对立事件**: A∪B=Ω, AB=∅, P(A)=1-P(B)
- **互不相容**: AB=∅ → P(AB)=0
- **独立**: P(AB)=P(A)P(B)

> ⚠️ 互不相容 ≠ 独立（互斥反而更不独立）

### 1.3 古典概型

- **公式**: P(A)=有利事件数/总事件数
- **排列**: A(n,m)=n!/(n-m)!
- **组合**: C(n,m)=n!/[m!(n-m)!]

## 二、随机变量分布

### 2.1 分布函数性质

1. 0≤F(x)≤1
2. 单调不减（不是严格增！）
3. 右连续: F(x+)=F(x)
4. P{X=x}=F(x)-F(x-)

### 2.2 常见分布期望方差（必背！）

| 分布 | 记号 | 期望 E(X) | 方差 D(X) |
|------|------|-----------|-----------|
| 0-1分布 | B(1,p) | p | p(1-p) |
| 二项分布 | B(n,p) | np | np(1-p) |
| 泊松分布 | P(λ) | λ | λ |
| 均匀分布 | U[a,b] | (a+b)/2 | (b-a)²/12 |
| 指数分布 | Exp(λ) 或 e(λ) | 1/λ 或 θ | 1/λ² 或 θ² |
| 正态分布 | N(μ,σ²) | μ | σ² |
| 伽马分布 | Γ(α,β) | αβ | αβ² |

> ⚠️ 指数分布参数注意：λ率参数（E=1/λ）或θ尺度参数（E=θ）

### 2.3 随机变量函数分布

**步骤**:
1. 求分布函数: F_Y(y)=P{Y≤y}=P{g(X)≤y}
2. 求导得密度: f_Y(y)=dF_Y(y)/dy

## 三、二维随机向量

### 3.1 联合分布

- **边缘分布**: P(X=x)=Σ P(X=x,Y=y)
- **联合密度**: f(x,y)=f_X(x)f_Y(y) （独立时）
- **边缘密度**: f_X(x)=∫ f(x,y) dy

### 3.2 独立性判定

**离散型**: P(X=x,Y=y)=P(X=x)·P(Y=y) 对所有x,y

**连续型**: f(x,y)=f_X(x)·f_Y(y) 对所有x,y

> ⚠️ 只要有一点不满足就不独立

### 3.3 二维正态分布

(X,Y)~N(μ₁,μ₂,σ₁²,σ₂²,ρ)

- ρ=0 ⇔ X,Y独立（仅二维正态成立！）
- X,Y线性组合仍服从正态分布

## 四、数字特征

### 4.1 期望性质

- E(c)=c
- E(aX+b)=aE(X)+b
- E(X+Y)=E(X)+E(Y) （无条件成立）
- E(XY)=E(X)E(Y) （仅当独立或不相关）
- E(X²)=[E(X)]²+D(X) （方差定义）

### 4.2 方差性质

- D(c)=0
- D(aX+b)=a²D(X) （常数不影响方差）
- D(X+Y)=D(X)+D(Y)+2Cov(X,Y)
- D(X-Y)=D(X)+D(Y)-2Cov(X,Y)

> ⚠️ D(X+Y)=D(X-Y) ⇔ Cov(X,Y)=0 ⇔ 不相关

### 4.3 协方差

**定义**: Cov(X,Y)=E(XY)-E(X)E(Y)

**性质**:
- Cov(X,X)=D(X)
- Cov(X,Y)=Cov(Y,X)
- Cov(aX,bY)=ab·Cov(X,Y)
- Cov(X+Y,Z)=Cov(X,Z)+Cov(Y,Z)
- X,Y独立 → Cov(X,Y)=0

> ⚠️ Cov里加常数直接删！（和方差一样）

### 4.4 相关系数

ρ_XY = Cov(X,Y)/√(D(X)D(Y))

**性质**:
- -1 ≤ ρ ≤ 1
- ρ=±1 ⇔ Y=aX+b 完全线性相关
- ρ=0 ⇔ 不相关
- ρ(aX+b,cY+d)=sign(ac)·ρ(X,Y)

### 4.5 不相关与独立

| 关系 | 结论 |
|------|------|
| 不相关 ⇔ Cov(X,Y)=0 ⇔ E(XY)=E(X)E(Y) |
| 独立 → 不相关 | 成立 |
| 不相关 → 独立 | ❌ 一般不成立 |
| 二维正态：不相关 ⇔ 独立 | 成立 |

## 五、大数定律与中心极限定理

### 5.1 伯努利大数定律

频率收敛于概率: S_n/n → p

### 5.2 辛钦大数定律

样本均值收敛于总体均值: X̄ → E(X)

### 5.3 棣莫弗-拉普拉斯定理（二项分布中心极限定理）

X~B(n,p)，n充分大时:
- X ≈ N(np, np(1-p))
- 标准化: (X-np)/√(np(1-p)) ≈ N(0,1)

### 5.4 独立同分布中心极限定理

X₁,...,X_n 独立同分布，E(X)=μ，D(X)=σ²:
- ΣX_i ≈ N(nμ, nσ²)
- X̄ ≈ N(μ, σ²/n)

**应用**: 涉及"和"或"平均"时使用

## 六、抽样分布与统计量

### 6.1 三大分布（必背！）

#### χ²分布（卡方分布）

**定义**: X₁,...,X_n 独立同分布于 N(0,1) → ΣX_i² ~ χ²(n)

**性质**:
- E(χ²(n))=n, D(χ²(n))=2n
- 可加性: χ²(n₁)+χ²(n₂)~χ²(n₁+n₂)
- 自由度=平方项数

#### t分布

**定义**: X~N(0,1), Y~χ²(n), X,Y独立 → X/√(Y/n) ~ t(n)

**性质**:
- E(t(n))=0 (n>1)
- 密度为偶函数，关于0对称
- n→∞时趋于N(0,1)

#### F分布

**定义**: X~χ²(n₁), Y~χ²(n₂), X,Y独立 → X/n₁ / Y/n₂ ~ F(n₁,n₂)

**性质**:
- 先分子后分母写自由度
- F~F(n₁,n₂) → 1/F~F(n₂,n₁)

### 6.2 正态总体抽样分布定理（核心！）

设 X₁,...,X_n 来自 N(μ,σ²)，样本均值 X̄，样本方差 S²:

| 结果 | 分布 |
|------|------|
| X̄ ~ N(μ, σ²/n) | 正态 |
| √n(X̄-μ)/σ ~ N(0,1) | 标准正态 |
| (n-1)S²/σ² ~ χ²(n-1) | 卡方(n-1) |
| X̄ 与 S² 独立 | 重要结论 |
| √n(X̄-μ)/S ~ t(n-1) | t分布 |

### 6.3 样本均值方差公式

- E(X̄)=μ
- D(X̄)=σ²/n
- E(S²)=σ²（无偏估计）

## 七、参数估计

### 7.1 矩估计

**步骤**:
1. 计算总体矩 E(X)（含参数θ）
2. 令 E(X)=X̄（样本矩）
3. 解出 θ̂

**公式**: K阶矩 → 令 E(X^K)=A_K（样本K阶矩）

### 7.2 最大似然估计

**离散型步骤**:
1. 构造似然函数: L(θ)=∏P(X_i)
2. 取对数: lnL(θ)
3. 求导: dlnL/dθ=0
4. 解出 θ̂

**连续型步骤**:
1. L(θ)=∏f(X_i)
2. lnL(θ)
3. 求导=0
4. θ̂

> ⚠️ 小写x改大写X得估计量

### 7.3 无偏性

**定义**: E(θ̂)=θ

**常用结论**:
- X̄ 是 μ 的无偏估计
- S² 是 σ² 的无偏估计
- 样本方差 S₀²=nΣ(X_i-X̄)²/n 是有偏估计

### 7.4 有效性

无偏估计中方差越小越有效

### 8.1 μ的置信区间

**σ已知**:
```
置信区间: X̄ ± z_α/2 · σ/√n
```

**σ未知**:
```
置信区间: X̄ ± t_α/2(n-1) · S/√n
```

### 8.2 σ²的置信区间

```
置信区间: [(n-1)S²/χ²_{1-α/2}(n-1), (n-1)S²/χ²_{α/2}(n-1)]
```

## 十、常见题型速解

### 题型1: 条件概率与全概率

**方法**: 列出完备事件组，用全概率公式

### 题型2: 联合分布求边缘/条件

**方法**: 
- 边缘: 横向/纵向求和（离散），积分（连续）
- 条件: P(Y=y|X=x)=P(X=x,Y=y)/P(X=x)

### 题型3: 求期望方差协方差

**步骤**:
1. 先求边缘分布
2. E(X)=Σx·P(X=x) 或 E(X)=∫x·f(x)dx
3. D(X)=E(X²)-[E(X)]²
4. Cov(X,Y)=E(XY)-E(X)E(Y)

### 题型4: 独立性与相关性判定

**独立性**: 检查 f(x,y)=f_X(x)f_Y(y) 是否对所有点成立

**相关性**: 计算 Cov(X,Y)，Cov=0则不相关

### 题型5: 中心极限定理应用

**识别**: n次独立重复、涉及"和"或"平均"

**步骤**:
1. 识别分布（通常是二项B(n,p）)
2. 计算E=np, D=np(1-p)
3. 标准化: (X-np)/√(np(1-p)) ≈ N(0,1)
4. 查表计算概率

### 题型6: 正态总体抽样分布

**识别**: 看样本均值X̄、样本方差S²、含σ²的式子

**关键**: (n-1)S²/σ² ~ χ²(n-1)

### 题型7: 矩估计与MLE

**矩估计**: E(X)=X̄，解方程

**MLE**: L(θ) → lnL → 求导=0 → θ̂

### 题型8: 区间估计

**公式代入**: 直接代入置信区间公式

## 十二、易错点总结

1. ❌ 分布函数是单调不减，不是严格增
2. ❌ 互不相容≠独立（互斥时P(AB)=0，若P(A)>0,P(B)>0则P(AB)<P(A)P(B），更不独立）
3. ❌ 不相关推不出独立（除非二维正态）
4. ❌ E(XY)=E(X)E(Y)仅当独立或不相关成立
5. ❌ 方差公式D(X)=E(X²)-[E(X)]²，不是[E(X)]²
6. ❌ 样本方差S²自由度是n-1，不是n
7. ❌ χ²分布自由度=平方项数（不是样本数）
8. ❌ 协方差和方差中加常数不影响，要删除
9. ❌ 指数分布参数λ和θ要注意（E=1/λ或E=θ）
10. ❌ MLE求导后要验证极值点
